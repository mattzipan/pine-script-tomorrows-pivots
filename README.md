# Pine Script Tomorrow's Pivots

A TradingView PineScript indicator that provides dynamic pivot point projections for both daily and weekly timeframes.

## Features
- **Daily Pivot Projections:** Forecasts tomorrow's pivot, support (S1-S3), and resistance (R1-R3) levels based on today's live data.
- **Current Weekly Pivot:** Displays a solid white line representing the pivot point for the current week.
- **Weekly Pivot Forecast:** Projects a dotted white line for next week's pivot point based on the current week's price action.
- **200 EMA:** A light blue, 3px thick exponential moving average for trend identification.

## Visuals
- **Solid White (Thin):** Daily Pivot Forecast.
- **Gray (Thin):** Daily S/R Levels.
- **Solid White (Thick):** Current Weekly Pivot.
- **Dotted White (Thick):** Next Week's Forecasted Weekly Pivot.
- **Light Blue (Thick):** 200-period EMA.

## Installation
1. Copy the code from `tomorrows-pivots.pine`.
2. Open TradingView.
3. Open the **Pine Editor** at the bottom.
4. Paste the code and click **Add to Chart**.
